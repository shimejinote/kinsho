# Tree models

Kenney Nature Kit GLBs remain here as CC0 archives, but the live forest scene
now uses **procedural mid-poly trees** (`treeFactory.ts`) — denser canopies /
trunks that instance cheaply without shipping larger third-party packs.

| Source | Notes |
|--------|--------|
| `components/forest/treeFactory.ts` | pine / fir / oak / maple at runtime |
| Kenney `*.glb` | unused by scene; kept for optional experiments |
